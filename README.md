# Whispering

A beautiful and intuitive macOS application built with SwiftUI.

## Requirements

- macOS 13.0+
- Xcode 14.0+
- Swift 5.7+

## Installation

1. Clone the repository
2. Open `Whispering.xcodeproj` in Xcode
3. Build and run the project
